# MAUI 学习项目

## 支持平台	
	
- Android 5.0 (API 21) 或更高版本。
- iOS 11 或更高版本，使用最新版本的 Xcode。
- 使用 Mac Catalyst 的 macOS 10.15 或更高版本。
- 使用 Windows UI 库 (WinUI) 3 Windows 11和Windows 10版本 1809 或更高版本。

  特别注意 MAUI <font color='red'><b>不支持</b></font> win7
